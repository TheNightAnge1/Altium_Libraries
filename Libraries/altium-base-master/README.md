# База библиотек Altium Designer

**/Symbols**  --  Условно-графические изображения компонентов  
**radiotech.DbLib**  --  Файл базы данных  
**radiotech_base.xls**  --  Файл БД в excel  
**/Templates**  --  Папка с шаблонами и форматками по ГОСТу
